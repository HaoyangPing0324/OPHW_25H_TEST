各位FPGA开发工程师及爱好者等，大家好：

首先感谢主办方能够给我这次试用测评OPHW-25H开源板的机会。该项目来源于小眼睛科技联合紫光同创及电子发烧友发起的# 拥抱开源！一起来做FPGA开发板活动。

以下为大家介绍本项目所有文件的组成：

## 项目文件结构

- **1_Demo**（所有实验的工程文件）
  - 重要声明：开发环境、特别鸣谢、版权声明
  - Basic：板卡基础模块测试
  - Advanced：图像视频处理
  - Demo目录

- **2_Hardware**（原理图及硬件使用手册）
  - 【OPHW_25H IO引脚分配总表】
  - 开发板PCB
  - 开发板原理图
  - 开发板硬件指导手册

- **3_IC_datasheet**（芯片手册）
  - 可能用到的芯片手册

- **4_Softwares**（软件）
  - 开发过程中可能用到的各种软件

- **5_Referring documents**（参考文档）
  - 所有demo的参考文档

- **6_doc**（开发文档）
  - 开发过程中可能用到的文档

- **7_Report**（测评报告）
  - 开箱测试、基础模块测试、图像视频处理

- **8_Video**（视频）
  - 开箱视频、板卡介绍、基础模块测试、图像视频处理

- **9_Others**（其他）
  - Log：问题、建议等

感谢大家耐心阅读，对于所给工程中不完善之处，敬请各位多提宝贵建议，谢谢！
