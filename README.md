## 开发板简介
OPHW-25H 开源板采用紫光同创 Logos-2 系列 FPGA（PG2L25H-6IMBG325）。高性能、全接口，满足图像处理、通信验证、AI 加速、数据采集等多种应用需求。

- **1_Demo**（所有实验的工程文件）

  - Advanced：图像视频处理
  - Demo目录

- **2_Hardware**（原理图及硬件使用手册）
  - 【OPHW_25H IO引脚分配总表】
  - 开发板PCB
  - 开发板原理图
  - 开发板硬件指导手册

- **3_IC_datasheet**（芯片手册）
  - 可能用到的芯片手册

- **4_Softwares**（软件）
  - 开发过程中可能用到的各种软件

- **5_Referring documents**（参考文档）
  - 所有demo的参考文档

- **6_doc**（开发文档）
  - 开发过程中可能用到的文档

- **7_Report**（测评报告）
  - 开箱测试、基础模块测试、图像视频处理

- **8_Video**（视频）
  - 开箱视频、板卡介绍、基础模块测试、图像视频处理

- **9_Others**（其他）
